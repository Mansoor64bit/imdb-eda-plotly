# IMDB Movie Dataset EDA using Plotly

## Overview

This project performs Exploratory Data Analysis (EDA) on an IMDB movie dataset using Python and Plotly.

The objective was to analyze movie trends, ratings, genres, and other important variables through interactive visualizations and data-driven insights.

---

## Tools Used

- Python
- Pandas
- NumPy
- Plotly

---

## Key Analysis Performed

- Data Cleaning
- Missing Value Analysis
- Rating Distribution Analysis
- Genre Analysis
- Revenue Analysis
- Trend Identification
- Interactive Visualizations

---

## Dataset

The dataset contains movie-related information such as:

- Movie Title
- Genre
- IMDB Rating
- Revenue
- Votes
- Release Year

---

## Project Workflow

1. Import Dataset
2. Data Cleaning
3. Exploratory Data Analysis
4. Visualization using Plotly
5. Interpretation of Results
